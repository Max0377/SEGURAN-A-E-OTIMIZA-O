OTIMIZAÇÃO E LIMPEZA INTERNA DO WINDOWS (SOFTWARE) COM FERRAMENTAS NATIVAS

🔹 1. Limpeza de Arquivos Temporários (%temp% e temp)

Como fazer:
Windows + R → digite %temp% → pressione Enter
Selecione todos os arquivos → Shift + Delete
Repita o processo digitando temp (sem %) e apague tudo novamente

Importância: Remove arquivos residuais gerados por programas, atualizações e o próprio sistema. Pode recuperar vários GBs de espaço.

🗑️ 2. Esvaziar a Lixeira

Como fazer:
Clique com o botão direito na Lixeira → Esvaziar Lixeira

Dica: Arquivos excluídos ainda ocupam espaço se não forem removidos da lixeira.

🧹 3. Limpeza de Disco (cleanmgr)

Como fazer:
Windows + R → digite cleanmgr
Selecione o disco C: (ou principal), marque tudo que for seguro e clique em OK

Dica avançada: Use "Limpar arquivos do sistema" para apagar atualizações antigas e arquivos de instalação.

⚙️ 4. Otimizar e Desfragmentar Unidades

Acesse o menu iniciar → digite "Desfragmentar" → clique em “Desfragmentar e Otimizar Unidades”
Selecione o disco e clique em “Otimizar”

Importância: Em HDDs melhora o acesso aos arquivos. Em SSDs, o TRIM é aplicado automaticamente.

🚫 5. Desabilitar Programas de Inicialização

Como fazer:
Ctrl + Shift + Esc → aba Inicializar → clique com o botão direito nos programas → Desabilitar

Exemplos: Spotify, Adobe Updater, Discord, Steam, OneDrive

📡 6. Desativar Aplicativos em Segundo Plano

Configurações → Aplicativos → Aplicativos em segundo plano
Vantagem: Reduz o consumo de memória e CPU.

⛔ 7. Finalizar Processos Pesados Manualmente

Como fazer:
Ctrl + Shift + Esc → aba Processos → ordene por CPU ou Memória → clique com botão direito e selecione “Finalizar tarefa”

Atenção: Evite encerrar processos do sistema.

🧠 8. Ajustar Efeitos Visuais

Windows + R → SystemPropertiesPerformance
Selecione "Ajustar para obter um melhor desempenho" ou escolha manualmente o que desativar

📅 9. Desativar Tarefas Agendadas

Windows + R → digite taskschd.msc
Acesse Biblioteca do Agendador de Tarefas → clique com botão direito → Desativar

🧰 10. Desinstalar Programas Inúteis

Windows + R → digite msi.cpl
Ordene por Tamanho ou Data e desinstale programas não utilizados

🧼 11. Sensor de Armazenamento (Storage Sense)

Configurações → Sistema → Armazenamento → Ativar Sensor de Armazenamento
Configure para excluir arquivos temporários, downloads antigos e esvaziar a lixeira automaticamente

📊 12. Usar o Monitor de Recursos (resmon)

Windows + R → digite resmon
Visualize os processos que mais consomem CPU, Disco, Memória e Rede

🛠️ 13. Verificar Inicialização Oculta (MSConfig)

Windows + R → digite msconfig
Na aba Serviços, marque “Ocultar serviços Microsoft” → desative os desnecessários

🔍 14. Verificar Corrupções com SFC

Abrir o Prompt de Comando como administrador e digitar:
sfc /scannow

O sistema fará a verificação e corrigirá arquivos corrompidos.

🔄 15. Reiniciar o PC com Frequência

Mesmo com o uso de suspensão, reiniciar libera memória e atualiza processos.
Recomendação: reiniciar ao menos 1 vez por dia.

🔧 16. Manter o Windows Atualizado

Configurações → Windows Update → Verificar atualizações

Motivo: Atualizações corrigem falhas, melhoram o desempenho e aumentam a segurança.
